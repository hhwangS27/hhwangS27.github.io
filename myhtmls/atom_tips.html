atom tips

ctl+click, multiple cursor

ctl+shift+p, you open the control palette.

ctl+t, search the file in a fuzzy way. very convinient to use.

file->stylesheet, this is the css that define the apprearance of atom itself. You can press ctl+shift+i to see the style justlike in chrome.

You can scroll pass the end of the file. Set in the editor to enable this.

You can show the indent guide. Set in the editor to enable this.

atom package

doge: after you type in "doge" and move your cursor to the left of "doge", you will see the doge meme..

pigment: show your css color in the color directly, after some setting the color will becomes a dot after the color which is great.

emmet: Type "!" then press Tab, you get standard html frame. Type "h2", you get "

<h2></h2>"

vim: you get vim mode.

minimap: you get the sublime-like minimap.
