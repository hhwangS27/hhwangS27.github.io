what is id_rsa?

The id_rsa can make you push without manually authorize everytime. When you put the id_rsa to a computer, that computer can edit your project as you are in the original place. When you generate your id_rsa, you can find a id_rsa.pub file in your .ssh folder. Paste all the content in that field to the github, with which github can use it to authorize your your local machine coordinating with the id_rsa file in your local machine.

How to add a new project to github

Create a new repository in github. Then type those commands below in git bash.

1.command: git remote add origin git@github.com:yueyueyueyue/basic_cpp.git

2.command: git add -A

3.command: git commit -m "add all origin files"

4.command: git push -u origin master

5. all done!
